# Linkit Menu Bar Animation

`linkit-menubar-animation-concept.png` was generated with the built-in `$imagegen`
workflow as a style reference for the animated macOS status icon.

The production menu-bar animation is rendered deterministically in Swift instead
of using AI-generated frame-by-frame art. That keeps the phone on the left, the
Mac on the right, and the center chain perfectly stable at status-bar sizes.

Prompt summary:

> Monochrome macOS menu-bar icon concept sheet for Linkit: Android phone fixed
> left, Mac laptop fixed right, chain link in the center, with connected,
> pairing, transfer, success, and broken-link/error states. Bold rounded strokes,
> high contrast, readable at 18-24 px, no text, no watermark.
